# DevOps Survival Guide: AI-Assisted Server Management & Deployment


Repository ini berisi kumpulan tugas dan latihan selama mengikuti program Sanbercode DevOps Bootcamp Batch 76. Berkaitan dengan proses pembelajaran dalam program DevOps Bootcamp yang berfokus pada pengelolaan server, automasi deployment, dan praktik DevOps modern. Saya adalah mahasiswa Sistem Informasi yang memiliki minat utama di bidang **Data Analysis**.
Melalui pembelajaran ini, saya mencoba memahami dasar-dasar DevOps untuk mendukung proses pengolahan, deployment, dan manajemen data secara lebih efisien.


---

## Deskripsi

Program ini berfokus pada pemahaman pengelolaan server dan automasi deployment.
Meskipun bukan fokus utama saya, materi yang dipelajari tetap relevan dalam konteks data, terutama untuk:

* Menjalankan proses data di server
* Mengelola environment untuk analisis data
* Mengotomatisasi workflow sederhana
* Memahami bagaimana data pipeline dapat dijalankan di lingkungan nyata

Materi juga mencakup penggunaan AI sebagai tools untuk membantu troubleshooting dan scripting 

---

## Relevansi dengan Data Analyst

Beberapa hal yang saya pelajari dan relevan dengan bidang data:

* Automasi proses menggunakan scripting (berguna untuk ETL sederhana)
* Pemahaman server sebagai tempat menjalankan proses data
* Version control untuk mengelola project data
* Pemanfaatan AI untuk meningkatkan efisiensi analisis dan debugging

---

## 🛠️ Tools & Teknologi

* Git & GitLab
* Linux / Command Line

* AI Tools (ChatGPT, Gemini, Copilot, dll)

## Insight yang Didapat

Dari pembelajaran ini, saya memahami bahwa:

* Data tidak hanya diolah di local, tetapi sering dijalankan di server
* Automasi sangat penting untuk workflow data yang berulang
* Version control membantu menjaga konsistensi project
* DevOps memiliki peran penting dalam mendukung proses data di skala yang lebih besar
